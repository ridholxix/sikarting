## Tentang Sikarting

Sikarting adalah aplikasi Sistem Pakar diagnosa dini Stunting yang menggunakan algoritma Naive Bayes dan Dempster Shafer dalam proses pengambilan keputusan (diagnosa). Aplikasi ini dirancang dalam rangka penyelesaian penelitian dengan judul "Naive Bayes dan Dempster Shafer untuk deteksi dini Stunting".

Aplikasi Sistem Pakar Stunting (Sikarting) menggunakan Framework laravel sebagai Backend Framework dan Bootstrap sebagai library Frontend. Aplikasi ini memiliki fitur CRUD, sistem autensitikasi, sistem diagnosa dan juga pemaparan langkah-langkah perhitungan algoritma (hasil diagnosa).
